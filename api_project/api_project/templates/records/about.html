About This Website

Welcome to Luis' Web Site! This web application is built using Django.

What is this website?

This website provides an academic course curriculum table, a rewards table and an API to access records on both tables.

How to use the API?

To use the API, you need an authentication token.

Steps to Get Your API Key:

  1. Register for an account.
  2. Login to your account.
  3. Navigate to the API Key page.
  4. Copy your personal API token.
  5. Use it in your API requests.

Example API Request Using cURL:

curl -H "Authorization: Token YOUR_API_KEY" https://laacdm.pythonanywhere.com/api/records/

Only authenticated users can access the API.
